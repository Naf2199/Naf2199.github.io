HelloWorld

Zhou Yufan 2021-10-02 16:21:28
Categories： 01 JAVA > 01 入门 Tags： Java

步骤

  1. 新建文件夹，存放代码
  2. 新建java文件
    • 文件后缀名为.java
  3. 编写代码
  4. 编译javac 文件名.java，生成一个class文件
  5. 运行class文件java 文件名
D:\Undergraduate\Code\JavaCode>javac HelloWorld.java

D:\Undergraduate\Code\JavaCode>java HelloWorld
Hello, World!

代码

public class HelloWorld{
	public static void main(String[] args) {
		System.out.print("Hello, world!");
	}
}

注意事项

  1. 注意单词大小写，标点符号中英文
  2. 尽量使用英文
  3. 文件名和类名必须保证一致，并且首字母大写
