(二) 标识符

​ Java所有的组成部分都需要名字。类名、变量名以及方法名都被称为标识符。

​ 注意:

  • 所有的标识符都应该以字母 ( A-Z / a-z ) 、美元符 ( $ ) 或者下划线 ( _ ) 开始
  • 首字符之后可以是字母 ( A-Z / a-z ) 、美元符 ( $ ) 、下划线 ( _ ) 或者数字的任何字符组合
  • 不能用关键字作为变量名或方法名
  • 标识符是大小写敏感的
  • 可以使用中文命名，但一般不建议这样去使用，也不建议使用拼音

1. 数值类型

    • int 占4个字节范围 : -2147483648 ~ 2147483647

      • long 类型要在数字后面加个 L

    1bit 表示1位

    1Byte 表示一个字节 1B = 8b

    1024B = 1KB

三、类型转换

  • 由于Java是强类型语言，所以要进行某些运算的时候，需要用到类型转换

    低 -------------------------------------------------> 高
  byte, short, char -> int -> long -> float -> double

  • 运算过程中，不同类型的数据要先转化为同一类型，然后进行运算

  • 低转高自动转换，高转低强制转换即(类型)变量名

注意

  1. 不能对布尔值进行转换

  2. 不能把对象类型转换为不相干的类型

  3. 在把高容量转换到低容量的时候，强制转换

  4. 转换的时候可能会存在内存溢出或者精度问题

    System.out.println((byte)128);  // -128  溢出
System.out.println((int)23.7);  // 23
System.out.println((int)-45.89F);  // -45
===========================================
char ch1 = 'a';
int num = ch1 + 1;
System.out.println(num);  // 98
System.out.println((char)num);  // b

// 操作比较大的数的时候，注意溢出问题
// JDK7新特性: 数字之间可以用下划线分割
int money = 10_0000_0000;
int years = 20;
int total = money * years;  // -1474836480  计算时溢出
long total2 = money * years;  // -1474836480  默认是int，转换之前已经存在问题了
long total3 = money * ((long)years);  // 20000000000  先把任意一个数转换为long即可

四、变量、常量

(一) 变量

  • 变量就是可以变化的量
  • Java是一种强类型语言，每个变量都必须声明其类型
  • Java变量是程序中最基本的存储单元，其要素包括变量名、变量类型和作用域
  • 不建议在一行中定义多个值

注意:

  • 每个变量名都有类型，类型可以是基本类型，也可以是引用类型
  • 变量名必须是合法的标识符
  • 变量声明是一条完整的语句，因此每一个声明都必须以分号结束

变量作用域

  • 类变量

  • 实例变量: 从属于对象，如不自行初始化，会输出这个类型的默认值

  • 局部变量: 必须声明和初始化值

    public class Demo05 {
    static double salary = 2500;  // 类变量 static
    String str = "hello world";  // 实例变量
    String name;  // 实例变量
    int age;  // 实例变量
    double num;  // 实例变量
    boolean flag;  // 实例变量

    // 其它方法
    public void method(){
        int i = 0;  // 局部变量
    }

    // main方法
    public static void main(String[] args) {
        // 实例变量
        // 变量类型 变量名称 = new Demo05();
        Demo05 demo05 = new Demo05();
        System.out.println(demo05.age);  // 0
        System.out.println(demo05.name);  // null  除了基本类型，都是"null"
        System.out.println(demo05.num);  // 0.0
        System.out.println(demo05.flag);  // false  0

        // 类变量
        System.out.println(salary);  // 2500.0
    }
}

(二) 常量(final)

  • 常量就是初始化后不能改变的量

  • 所谓常量可以理解为一种特殊的变量，它的值被设定后，在程序运行过程中不能被改变

  • 常量名一般使用大写字母

    public class Demo06 {
    // 修饰符不存在先后顺序  修饰符: 变量类型前的关键字
    // final 常量名 = 值;
    static final double PI = 3.14159;
    
    public static void main(String[] args) {
        System.out.println(PI);
    }
}

(三) 变量的命名规范

  • 所有的变量、方法、类名：见名知意
  • 类成员变量：首字母小写和驼峰原则 : monthSalary
  • 局部变量：首字母小写和驼峰原则
  • 常量：大写字母和下划线 : MAX_VALUE
  • 类名：首字母大写和驼峰原则 : Man，GoodMan
  • 方法名：首字母小写和驼峰原则 : run()，runRun()

(二) 自增自减运算符、初识Math类

public class Demo04 {
    public static void main(String[] args) {
        // ++ 自增 -- 自减  一元运算符
        int a = 3;
        int b = a++;  // 先 b = a = 3，再 a = a+1 = 4
        int c = ++a;  // 先 a = a+1 = 5，再 c = a = 5

        System.out.println(a);  // 5
        System.out.println(b);  // 3
        System.out.println(c);  // 5

        // Math类
        // 幂运算 2^3 2*2*2 = 8
        // 很多运算我们会使用一些工具类来进行操作
        double pow = Math.pow(2, 3);

        System.out.println(pow);  // 8.0
    }
}

(三) 逻辑运算符、位运算符、条件运算符

1. 逻辑运算符

  • 逻辑与运算 ( && ) : 两个变量都为真，结果才为真

  • 逻辑或运算 ( || ) : 两个变量中有一个为真，结果就为真

  • 逻辑非运算 ( ! ) : 如果是真，结果为假；如果是假，结果为真

    public class Demo05 {
    public static void main(String[] args) {
        // &&(and)  ||(or)  !(取反)
        boolean a = true;
        boolean b = false;

        System.out.println("a && b: " + (a && b));  // a && b: false
        System.out.println("a || b: " + (a || b));  // a || b: true
        System.out.println("!(a && b): " + !(a && b));  // !(a && b): true

        // 短路运算
        int c = 5;
        boolean d = (c<4) && (c++<4);

        System.out.println(d);  // false
        System.out.println(c);  // 5  "c++"没有运行
    }
}

2. 位运算符

  • 二进制运算

    A = 0011 1100
B = 0000 1101

A&B = 0000 1100  // 同1为1，否则为0
A|B = 0011 1101  // 有1为1，否则为0
A^B = 0011 0001  // 相同为0，不同为1 即不进位的二进制加法
~B = 1111 0010  // 相反

按位左移  --  <<
    i << 1 表示把 i 的所有二进制位左移一位，右边补0
    左移 n 位相当于乘以2的n次方【二进制，十进制底为10】
    2*8 = 16  2 *2*2*2 即 2<<3
按位右移  -- >>
   i >> 1 表示把 i 的所有二进制位右移一位，左边补0【正数/无符号右移">>>"】或1【负数】
    右移 n 位相当于除以2的n次方【二进制】（前提是数据不能丢失）
0000 0000    0
0000 0001    1
0000 0010    2
0000 0100    4
0000 1000    8
0001 0000    16

六、包机制、JavaDoc

(一) 包机制

  • 为了更好地组织类，Java提供了包机制，用于区别类名的命名空间

  • 包机制的语法格式为 :

    package pkg1[. pkg2[. pkg3...]];

  • 一般利用公司域名倒置作为包名 io.github.naf2199

  • 为了能够使用某一个包的成员，我们需要再Java程序中明确导入该包，使用import语句可以完成此功能

    import package1[.package2...].(classname | *); // *通配符，导入这个包下所有的类
