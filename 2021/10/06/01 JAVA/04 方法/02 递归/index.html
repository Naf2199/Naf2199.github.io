02 递归

Zhou Yufan 2021-10-06 11:29:36
Categories： 01 JAVA > 04 方法 Tags： Java

  • 递归就是自己调用自己

  • 利用递归可以用简单的程序来解决一些复杂的外提，它通常把一个大型复杂的问题层层转化为一个与原问题相似的规模较小的问题来求解，递归策略只需少量的程序就可以描述出解题过程所需要的多次重复计算，大大减少了程序的代码量，递归的能力在于用有限的语句来定义对象的无限集合

  • 递归结构包括两个部分：

    • 递归头：什么时候不调用自身方法。如果没有头，将陷入死循环
    • 递归体：什么时候需要调用自身方法

  • 阶乘

    public class Demo05 {
    public static void main(String[] args) {
        System.out.println(factorial(5));  // 120
    }

    // 阶乘
    public static int factorial(int n) {
        if (n == 1){
            // 递归头
            return 1;
        } else {
            // 递归体
            return n * factorial(n - 1);
        }
    }
}

  • 能不用递归就不用递归
